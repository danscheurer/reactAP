Ejercicio 1

Enunciado:

Tengo desarrollada la siguiente función en Javascript:

function suma (a,b,callback){ let c = a +b; callback();
}

¿Que es el tercer parámetro recibido?
¿En que casos es obligatorio desarrollar este tipo de funciones?
Realizar un llamado a la función de ejemplo.

Respuesta:

El tercer parametro recibido es una funcion de "callback".
Si queremos que se realice una tarea extra luego de que se complete una función, es obligatorio su uso.


El llamado a la función se encuentra en el archivo .js enlazado.
